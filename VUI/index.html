Student Voice Assitant

Voice User Interface for Computer Science Students

Possible Utterances:
  • Greetings
  • -o- You can say a greeting (e.g. "Hello", "Hi", "What's up")
  • -o- You can also say thank you after a response

  • Commands
  • -o- Search for definition of a word
  • -o- Search for the Antonym or synonym of a word
  • -o- Ask to learn a programming language. (e.g. Javascript, Python)
  • -o- Ask about the referencing style accepted by Middlesex University
  • -o- Ask about a computer science module / course.
  • -o- Ask about the structure of your dissertation

Note:You must always click the button per question/statement
